## Ⅰ. 송길영 박사 핵심 주장

### 1. 핵심 선언: "대마불사 → 대마필사"

| 과거 (중량문명) | 현재 (경량문명) |
|----------------|-----------------|
| **대마불사(大馬不死)** | **대마필사(大馬必死)** |
| "크면 안전하다" | "크면 죽는다" |
| 규모의 경제가 생존 보장 | 속도와 유연성이 생존 결정 |

### 2. 경량문명의 두 축

#### 🧠 지능의 범용화
- 과거: 고차원 업무 → 대규모 조직/전문가 집단 필요
- 현재: AI가 '부지런한 지능' + '초월적 지능' 제공
- 결과: 개인/소규모 팀도 고차원 업무 실행 가능

#### 🤝 협력의 경량화
- 과거: 피라미드형 위계, 관료제, 영구 고용
- 현재: 클러스터(소규모 모둠), 프로젝트 단위 협력
- 원칙: "지금 만납니다. 잠시 만납니다. 다시 만납니다."

## Ⅱ. 경량문명 조직의 특징

### 조직 구조 변화

| 구분 | 중량문명 | 경량문명 |
|------|----------|----------|
| **조직 형태** | 피라미드형 위계 | 클러스터(소규모 모둠) |
| **협력 방식** | 영구 고용, 평생직장 | 단속적 협력, 프로젝트 기반 |
| **의사결정** | 다단계 결재 | 의사결정 단계 축소 |
| **가치 평가** | 매출, 인원수 | 인당 시총, 인당 매출 |
| **핵심 역량** | 관리 능력, 조직 경험 | AI 활용 능력, 결과/역량 |

### 프로세스 변화 예시: 광고업계

**Before (중량문명)**
```
광고주 → 대행사 → 협력사 → 매체 → 모델
```

**After (경량문명)**
```
광고주 → 모델 → 광고주
```

→ 중간 단계 급격히 축소, 3개월 만에 밸류체인 단순화

## Ⅲ. 경량문명 시대의 기업 생존 조건

### 핵심 인용
> "기업 규모가 크더라도 그 내부 구조가 성기고 밀도가 낮다면 날아오를 수 있다.  
> 작은 기업일지라도 문화와 태도, 구조가 무겁다면 가라앉을 것이다.  
> **돌멩이는 작아도 가라앉고, 비행기는 거대하지만 하늘 위를 뜬다.**  
> 규모가 아니라 구조의 문제다."  
> — 송길영

### 생존 조건 3가지

1. **빠른 의사결정**: 의사결정 속도가 생존을 결정
2. **민첩한 조직 구조**: 불필요한 단계 제거, 가볍게 움직임
3. **AI 활용 역량**: "AI가 못 하는 일을 증명해라" (쇼피파이 CEO)

## Ⅳ. 코닥 사례 분석

### 1. 코닥의 역사와 위상

| 연도 | 사건 |
|------|------|
| 1882년 | 조지 이스트만, 코닥 설립 |
| 1888년 | "당신은 찍기만 하세요" 광고 캠페인 |
| **1975년** | **세계 최초 디지털 카메라 개발** (스티브 새슨) |
| 1979년 | 래리 맷슨, "2010년대 디지털 전환 완료" 보고서 작성 |
| 1990년대 | 미국 필름 시장 점유율 90% |
| 1995년 | 기업가치 세계 4위 (코카콜라, 맥도널드, IBM 다음) |
| 1998년 | 일본 기업들 보급형 디지털카메라 출시 시작 |
| 2004년 | 필름 카메라 사업 포기 |
| 2007년 | 13개 공장 폐쇄, 5만명+ 해고 |
| **2012년** | **파산보호 신청** |

### 2. 실패 원인 분석

#### 🔴 핵심 실패 원인: 자기 잠식(Cannibalization) 회피

```
┌─────────────────────────────────────────────────────────────┐
│  1975년: 세계 최초 디지털 카메라 개발                          │
│  ↓                                                          │
│  경영진 판단: "디지털 카메라가 필름 시장을 잠식할 것"              │
│  ↓                                                          │
│  결정: 상용화 중지, 디지털 출현을 의도적으로 억제                  │
│  ↓                                                          │
│  20년간 억제 전략 유지                                         │
│  ↓                                                          │
│  1998년: 일본 기업들 디지털 카메라 대중화                        │
│  ↓                                                          │
│  코닥, 후발주자로 전락                                         │
│  ↓                                                          │
│  2012년: 파산                                                │
└─────────────────────────────────────────────────────────────┘
```

#### 🔴 세부 실패 요인

| 요인 | 설명 |
|------|------|
| **기존 사업 보호** | 필름 사업의 높은 수익성 유지에 집착 |
| **변화 과소평가** | 디지털 기술 발전 속도와 영향력 과소평가 |
| **조직 저항** | 필름 중심 조직 문화가 새로운 기술 경시 |
| **특허만 보유** | 1,100개+ 디지털 이미지 특허 보유했으나 상용화 실패 |
| **늦은 전환** | 뒤늦게 Easy Share 전략 시도했으나 이미 시장 상실 |

### 3. 코닥 vs 후지필름 비교

| 구분 | 코닥 | 후지필름 |
|------|------|----------|
| **대응 전략** | 필름 시장 방어 | 사업 다각화 |
| **핵심 결정** | 디지털 상용화 억제 | 필름 기술 응용 (화장품, 의료) |
| **조직 변화** | 기존 구조 유지 | 적극적 구조조정 |
| **결과** | 2012년 파산 | 현재도 건재 |

## Ⅴ. 엔키아 적용 시사점

### 코닥 사례가 주는 교훈

| 코닥의 실수 | 엔키아 적용 질문 |
|-------------|------------------|
| 기존 필름 사업에 집착 | 기존 Enterprise Monitoring 사업에만 의존하고 있는가? |
| 디지털 카메라 기술 보유했으나 상용화 안함 | AI/자동화 기술을 제품화하고 있는가? |
| 조직이 변화를 거부 | 새로운 사업(보안 모니터링 등)을 위한 조직 체계가 있는가? |
| 경쟁자 움직임 과소평가 | 신규 진입자들의 AI 기반 솔루션을 모니터링하고 있는가? |

### 송길영 관점에서의 조직 변화 방향

```
[기존 조직 구조]                    [경량문명 조직 구조]
                                   
┌──────────────────┐              ┌──────────────────┐
│     경영진        │              │     경영진        │
├──────────────────┤              └────────┬─────────┘
│    중간관리자      │                      │
├──────────────────┤              ┌────────┴─────────┐
│    실무자         │              │   프로젝트 클러스터   │
└──────────────────┘              │   (Task 단위 협력)   │
                                  └──────────────────┘
다단계 결재                        의사결정 단계 축소
영구 고용                          프로젝트 기반 협력
```

### 프로세스 변경 포인트

회장님 메모에 있던 "프로세스 변경" 관련:

1. **의사결정 단계 축소**: 7단계 → 3~4단계 (현대모비스 사례)
2. **Task 단위 업무 분배**: 조직의 목표 = 조직 위치 상승 → **업무 완결**
3. **AI 협력 도입**: CHR(인적자원관리) → **CTO(Chief Task Officer)** 중심
4. **속도 중심 문화**: "중첩된 결재 단계와 위계는 속도를 늦춘다"

## Ⅵ. 핵심 메시지 (회장님 보고용)

### 1장 요약 키워드

> **"경량문명으로의 전환은 선택의 문제가 아니다"** — 송길영

1. **코닥 교훈**: 기술은 있었으나 조직이 변하지 않아 망함
2. **핵심 전환**: 대마불사 → 대마필사 (크면 죽는다)
3. **생존 조건**: 규모가 아닌 구조의 문제
4. **조직 방향**: 가볍고, 빠르고, 유연한 구조
