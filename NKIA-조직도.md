## 주요 제품 라인업

| 제품 | 설명 |
|------|------|
| **Polestar8** (EMS) | SMS, NMS, DPM, APM 모니터링 |
| **Polestar10** (통합신제품) | EMS(SMS, NMA, DPM, APM, K8s) + ITSM(개발중) + AUTOMATION(미개발) |
| **AUTOMATION** | 스크립트 기반 서버 자동화 |
| **ITSM** | IT 서비스 관리 |
| **ITAM** | IT 자산 관리 |

## 겸임 현황

- **이일섭**: 영업대표 + 영업본부장 + IoT사업본부장
- **진원경**: AI연구소장 + 전략기획실장
